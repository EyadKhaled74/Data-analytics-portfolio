# Sample Superstore Analysis Dashboard

This repository contains the Power BI deliverables for a Sample Superstore Analysis project completed during a Data Analyst internship at Uneeq Interns.

## Project Overview

The analysis evaluates sales performance, profitability, shipping behavior, and regional trends across categories and sub-categories.

## Key Insights

- Total Revenue: 2.30M
- Net Profit: 286K
- Average Profit Margin: 12.01%
- Top-performing sub-categories: Copiers, Phones, Accessories
- Tables had negative profitability
- West region had the strongest profit margin
- Central region underperformed
- Standard shipping was dominant; faster shipping options were underutilized

## Recommendations

- Improve pricing and discount strategy for low-profit products, especially Tables
- Prioritize regional optimization in underperforming areas such as Central
- Promote high-margin products and improve shipping mix for better operational efficiency
