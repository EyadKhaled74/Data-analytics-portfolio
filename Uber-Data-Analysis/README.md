# Uber Data Analysis

This folder contains a Power BI dashboard project for Uber trip data analysis.
The visual story is organized across executive, mobility, and performance dashboard pages.

## Dashboard Pages

- Executive Hub: total trips, total miles, business percent, highest mph, average speed, and monthly mileage trends.
- Mobility Insights: total miles by time period, category analysis, business vs personal miles, and speed distribution.
- Performance & Efficiency: trip filters, average ratings, distance analysis, and operational efficiency metrics.
