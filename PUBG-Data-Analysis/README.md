# PUBG Data Analysis

This folder contains a Power BI dashboard project for PUBG weapon and performance analytics.
The dashboard highlights weapon efficiency, damage metrics, and usability insights.

## Dashboard Pages

- Meta Overview: weapon count, average DPS, max DPS, and top weapon efficiency metrics.
- Damage & Performance Analysis: DPS rankings, armor mitigation, magazine capacity, and arsenal breakdown.
- Weapon Mechanics & Usability: weapon damage, DPS, kill shot analysis, and range performance mapping.
